# 功能清单与测试规范

> 这份文档列出本项目所有面向用户的功能，每个功能给出：入口、行为约定、关键代码、测试用例。
> 以"功能为单元"组织，方便按版本回归。

## 0. 文档约定

### 0.1 编号规则
`F-<模块号>.<功能号>` 表示功能；`T-<模块号>.<功能号>.<用例号>` 表示该功能下的测试用例。

### 0.2 用例模板
| 字段 | 说明 |
| --- | --- |
| 前置条件 | 进入该用例前需要的状态（已登录/已添加书/某本书已打开等） |
| 操作步骤 | 1. ... 2. ... 3. ... |
| 预期结果 | 必须满足的可观察行为 |
| 优先级 | P0 阻断 / P1 主流程 / P2 次要 / P3 边角 |
| 设备 | iPhone Safari / iPhone Edge / 汉王电纸书 Via / 桌面 Chrome / 桌面 Edge / 全部 |

### 0.3 维护原则
- 每次新增/修改功能必须同步更新本文件，并补 P0/P1 用例
- 修复 bug 必须**先**在对应功能下补一条复现用例（保留原 bug 描述），再关联修复 commit
- 已废弃功能保留条目并标 `[Deprecated]`，便于追溯历史

## 1. 阅读核心 (Reading Core)

### F-1.1 文本渲染
**入口**：书架 → 双击书目 → 进入阅读视图
**核心代码**：`src/components/ReadingArea.tsx`、`src/lib/processBookContent.ts`

**行为约定**：
- 段落保留原文换行，每段间距固定
- 单词、所有格 `'s`、缩写、标点正确分词；`Cloister's` 应可点击 `Cloister`，`haven't` 整体不可点击但显示完整
- 大文档使用虚拟滚动（`@tanstack/react-virtual`）

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-1.1.1 | 打开 1MB+ 长文本 | 列表流畅滚动，无白屏卡顿 | P0 |
| T-1.1.2 | 点击 `Cloister's` 中的 `Cloister` | 弹出释义 tooltip | P1 |
| T-1.1.3 | 点击 `haven't` | 不弹 tooltip（整体作为缩写） | P2 |
| T-1.1.4 | 文本含 `[H2]Chapter[/H2]` | 渲染为 H2 章节标题样式 | P1 |

### F-1.2 翻页模式
**入口**：阅读 → 设置 → 翻页模式
**核心代码**：`ReadingArea.tsx::scrollReadingPage / computeSnappedPageScrollTop / updatePageMask`

**行为约定**：
- **滑动翻页**：使用浏览器原生竖向滚动；不响应水平手势、点击翻页、音量键
- **点击/手势翻页**：
  - 点击屏幕**左半空白**翻上一页，**右半空白**翻下一页；点击 `.word`/`.annotation`/`.sentence-annotation` 不翻页
  - 横向 swipe > 50px 翻页（左滑下一页 / 右滑上一页）
  - `AudioVolumeUp` ↑ 翻上一页，`AudioVolumeDown` ↓ 翻下一页
  - 翻页无动画；底部不完整行用与背景同色的遮罩盖住
  - 上下翻页之间不重复行也不漏行

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-1.2.1 | 点击屏幕右半空白 | 翻到下一页，首行不重复上一页 | P0 |
| T-1.2.2 | 点击屏幕左半空白 | 翻到上一页 | P0 |
| T-1.2.3 | 在分页模式点击单词 | 弹释义 tooltip，**不翻页** | P0 |
| T-1.2.4 | 在分页模式横向左滑 | 翻下一页 | P1 |
| T-1.2.5 | 在汉王电纸书按音量上键 | 翻上一页 | P1 |
| T-1.2.6 | 切换"滑动翻页"后用鼠标滚轮 | 应能滚动 | P0 |
| T-1.2.7 | 切换"滑动翻页"后用鼠标滚轮（**回归**） | 不应出现"切换后短时滑不动"的现象 | P1 |
| T-1.2.8 | 翻多页后查看底部 | 始终无半行字残留 | P1 |

### F-1.3 阅读进度
**入口**：自动；左下角 `xx.xx%`
**核心代码**：`ReadingArea.tsx`、`useBookshelf.ts::updateBookProgress`

**行为约定**：
- 滚动时实时更新百分比
- 重新打开同一本书自动恢复到上次位置
- 新加入的书 progress 默认为 0，不复用上一本书

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-1.3.1 | 滚动到 50% 后退出 → 重新打开 | 恢复到 50% | P0 |
| T-1.3.2 | 上一本书读到 100% → 添加新书并打开 | 新书在 0% 而非 100% | P1 |

### F-1.4 书签
**入口**：阅读视图工具栏 → 添加书签
**核心代码**：`useBookshelf.ts::addBookmark/removeBookmark`、`ReadingArea.tsx::addBookmarkFn`

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-1.4.1 | 添加书签 → 跳到其他位置 → 点击书签 | 跳回书签处 | P1 |

## 2. 词典与释义 (Dictionary)

### F-2.1 双层词典查找
**入口**：单击单词；自动触发
**核心代码**：`src/lib/dictionary.ts::smartLookup`、`src/lib/dictLoader.ts::smartLookupExternal`

**行为约定（保守策略）**：
1. 先查原词
2. 再查屈折变体（`-s/-es/-ies/-ed/-ing` 及双辅音还原与不规则动词/名词）
3. **不**做派生形态学（`-er/-est/-ly/-ness/-ment/-able/...`）
4. **不**做前缀剥离（`un-/in-/dis-/...`）
5. 都查不到返回 null，触发 AI 翻译兜底

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-2.1.1 | 查 `running` | 返回 run 的释义 | P0 |
| T-2.1.2 | 查 `oxen` | 返回 ox 的释义 | P1 |
| T-2.1.3 | 查 `tries` | 返回 try 的释义 | P1 |
| T-2.1.4 | 查 `pier`（dict 没有 pier 但有 pie）| **返回 null，由 AI 接管**，不能误返回"馅饼" | P0 |
| T-2.1.5 | 查 `unaging` | 不能返回 age 的"年龄" | P0 |
| T-2.1.6 | 查 `indiscernibly` | 不能返回 discern 的"辨别,识别" | P0 |

### F-2.2 单击查词（Tooltip）
**入口**：阅读时单击单词
**核心代码**：`WordTooltip.tsx`、`page.tsx::handleWordClick`

**行为约定**：
- 单击 300ms 后弹 tooltip（留出双击窗口）
- tooltip 显示：原词、词性、释义、词根（lemma）、发音按钮、标注按钮

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-2.2.1 | 单击词 → 等 0.3s | 显示 tooltip 含释义 | P0 |
| T-2.2.2 | 单击释义未知词（如 wobbling） | 显示"未找到释义" + AI 兜底（双击场景） | P1 |
| T-2.2.3 | 点击 tooltip 喇叭按钮 | 朗读该词 | P2 |

### F-2.3 双击自动标注
**入口**：阅读时双击单词
**核心代码**：`page.tsx::handleWordDoubleClick`

**行为约定**：依次尝试 内置词典 → 外部词典 → AI 翻译；成功后写入 `annotations`，并加入全局词汇表
- 当前 `dictMode`（zh/en/en-simple）决定释义语言
- **失败信息（如"翻译失败"）不应被写入词汇表**

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-2.3.1 | 双击未标注词 | 自动加注，全文同词形高亮 | P0 |
| T-2.3.2 | AI 翻译失败 | 不写入词汇表 | P0 |
| T-2.3.3 | 双击已标注词 | 不重复请求，不修改现有释义 | P1 |

### F-2.4 句子翻译
**入口**：选中一段文字（>1 词） → 选区面板 → 翻译
**核心代码**：`ReadingArea.tsx::handleTextSelection`、`page.tsx::translate`

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-2.4.1 | 选中两个词以上 | 弹翻译面板 | P1 |
| T-2.4.2 | 选中单个词 | 不弹翻译面板（应触发单词流程） | P1 |

### F-2.5 释义语言切换
**入口**：阅读视图顶部"中/英/简易英"
**核心代码**：`useBookshelf.ts::dictMode`、`mergedAnnotationsForRender`

**行为约定**：切换语言后已标注词的释义自动跟随；未本地化的词显示对应语言的占位符

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-2.5.1 | 在 zh 模式标注 100 词 → 切到 en | 已标注词显示英文释义（本地或 AI 补全） | P1 |
| T-2.5.2 | 词汇表面板"语言补全"按钮 | 显示进度条，逐词补全缺失语言 | P2 |

## 3. 词汇管理 (Vocabulary)

### F-3.1 全局词汇表
**入口**：右下侧栏 → 词汇本图标
**核心代码**：`GlobalVocabularyPage.tsx`、`VocabularySidebar.tsx`、`useBookshelf.ts::globalVocabulary`

**行为约定**：
- 同一 lemma 只保留一条
- 删除（垃圾桶）同时清理 lemma 与原词键
- 删除后该词记入 `masteredWords`（已掌握）

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-3.1.1 | 标注 `tea` 后再点垃圾桶 | 词汇表与侧栏中均消失，且不重复出现 | P0 |
| T-3.1.2 | 删除后再次双击同词 | 不会出现重复条目 | P1 |

### F-3.2 已掌握标记
**核心代码**：`useBookshelf.ts::masteredWords`

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-3.2.1 | 删除某词后打开词汇分析 | 该词标 ★（已掌握） | P1 |
| T-3.2.2 | 词汇分析"全选添加" | 已掌握/已存在的词不被再次添加 | P0 |

### F-3.3 词汇分析
**入口**：阅读视图工具栏 → 词汇分析
**核心代码**：`BookVocabAnalysis.tsx`

**行为约定**：
- 排除停用词、缩写碎片、长度过短/过长的词
- 支持按 CEFR 等级筛选（多选切换）、按词长/词频筛选、隐藏已加入/已掌握
- 100 词/页分页 + 回顶按钮
- 批量添加时缺失释义自动调用 AI

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-3.3.1 | 设置词频≥3 | 仅显示出现 3 次以上的词 | P1 |
| T-3.3.2 | 多选 A1 + A2 | 同时显示两级 | P1 |
| T-3.3.3 | 批量添加 50 词 | 进度条递进，缺失释义由 AI 补 | P1 |
| T-3.3.4 | 翻到第 2 页 | 后续 100 词正常显示 | P1 |

### F-3.4 CEFR 分级标注
**入口**：设置 → CEFR 自动标注 → 选 A1~C2
**核心代码**：`vocabLevel.ts`、`page.tsx`

## 5. TTS 朗读 (Text-to-Speech)

**入口**：阅读视图顶部 ▶ 按钮
**核心代码**：`useTTS.ts`、`api/tts/route.ts`、`api/tts-edge/route.ts`

### F-5.1 多源回退链
**行为约定（按设备优先级）**：
1. iPhone Safari/桌面 Chrome/桌面 Edge → Web Speech API（本地）
2. 汉王电纸书或 Web Speech 不可用 → Edge TTS（远程免费）
3. 兜底 → Coze TTS（远程付费）

iPhone 上提供"本地/在线"手动开关；可选择本地 voice。

### F-5.2 朗读高亮 + 滚动
**行为约定**：
- 从首屏可见首句开始
- 当前句标浅豆绿色 `#d4edda`
- 已读到当前页 80% 时再把下一段置顶（避免频繁跳动）
- 缩写如 `Mr. / Dr.` 不被句末断句

| 用例 | 步骤 | 预期 | 优先级 |
| --- | --- | --- | --- |
| T-5.2.1 | 在含 `Mr. Azik` 的段落朗读 | 不在 `Mr.` 处断句 | P0 |
| T-5.2.2 | 暂停后再点播放 | 从中断处继续，**不**重头 | P0 |
| T-5.2.3 | 一段最后一句 | 不被跳过 | P0 |
| T-5.2.4 | 移动端连读 5 句 | 无空字符串请求、不中断 | P1 |

### F-5.3 控制条
- 速度：下拉 / 滑动条
- 上一句 / 下一句
- 风格与目录按钮一致
